## Description

This repository implements the Dimension Expansion Network (DEN) for inverse metalens design.

Given a target focal position, DEN first transforms the low-dimensional target coordinates into a high-dimensional two-dimensional conditioning representation using Fourier feature encoding and outer-product expansion. The representation is then processed by an attention-based U-Net to predict the refractive-index distribution. The generated structure is evaluated using a differentiable full-wave finite-difference frequency-domain (FDFD) electromagnetic solver with Fourier-based free-space propagation, and gradients are back-propagated through the solver to train the network end-to-end.

## Demo

### Running the demo

The evaluation script loads the pretrained model at epoch 8,300 and evaluates metalens designs generated by the network over an 11 × 11 grid (121 target focal positions).

Because the metalens design is mirror symmetric with respect to the y axis, all 121 target focal positions are first evaluated. The better-performing design from each mirror-symmetric pair is retained, while the 11 focal positions on the symmetry axis are kept unchanged. Consequently, the script saves visualizations for **66 unique focal positions**, applying horizontal flipping when necessary for mirror-symmetric cases.

Run

```bash
python test.py
```

The expected output images are provided in

```text
expected_output/
```

These reference images were generated using the released pretrained checkpoint and can be used to verify correct installation and execution.

### Expected runtime

The neural network predicts a metalens structure in approximately **0.15 seconds** per target focal position on an NVIDIA A6000 GPU.

The subsequent full-wave electromagnetic simulation requires approximately **1.0 second** per focal position.

Without saving visualization figures, the complete evaluation of all **66** focal positions typically requires **about 2 minutes**.

Generating and saving all visualization figures increases the total runtime to approximately **15 minutes**, with the additional time dominated by figure rendering and disk I/O.

## Instructions for Use

### Training

To train the DEN from scratch:

```bash
python trainmetalens80by80.py
```

The script samples random focal positions at each iteration, runs differentiable full-wave electromagnetic simulations on GPU, and back-propagates through the solver. Model checkpoints are saved every 100 iterations to a timestamped output directory under `results_new_adjoint_fourier/`. Training progress (average figure of merit) is logged to `log.txt` in the same directory.

**Typical training time:** approximately 12 hours on a single NVIDIA A6000 GPU for **10,000 training iterations**.

The pretrained checkpoint included with this repository corresponds to **8,300 training iterations**, which was used for the experiments reported in the manuscript.
